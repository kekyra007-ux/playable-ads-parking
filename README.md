# Playable Ad — Parking

Интерактивный рекламный баннер (playable ad) с механикой паркинга автомобилей. Пользователь пальцем рисует маршруты для двух машин, после чего они анимированно проезжают к своим парковочным местам.

## Механика

- Два автомобиля (красный и жёлтый) стоят на сцене
- Игрок рисует свайпом путь для каждой машины до нужного парковочного слота
- После того как оба маршрута нарисованы, машины одновременно едут по ним
- По завершении появляется финальный экран с кнопкой CTA, ведущей на целевую страницу
- Если игрок бездействует — через 20 секунд демо запускается автоматически (hint-анимация)

## Стек

| Технология | Назначение |
|---|---|
| TypeScript | основной язык |
| Pixi.js v6 | 2D-рендеринг (WebGL / Canvas) |
| GSAP 3 | анимации машин и UI |
| Webpack 5 | сборка и оптимизация |
| html-inline-script-webpack-plugin | инлайн скриптов в единый HTML-файл |

## Структура проекта

```
src/
├── core/          # Game, Scene, GameplayController, типы
├── scenes/        # BootScene (загрузка), StartScene (геймплей)
├── entities/      # Car, ParkingSlot, HintHand
├── ui/            # FinalOverlay, PathDrawer
├── config/        # gameConfig, layout, path
└── utils/         # вспомогательные утилиты
```

## Сборка

```bash
# Установка зависимостей
npm install

# Продакшн-сборка (один HTML-файл с инлайн-скриптами)
npm run build
```

Результат сборки — единственный файл `dist/index.html`, готовый к загрузке на рекламные платформы.
